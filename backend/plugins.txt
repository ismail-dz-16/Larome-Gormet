------------------------------------------------------------

[STEP 1: ENABLE LEGACY API]
Go to: WooCommerce > Settings > Advanced > Legacy API.
Action: Check "Enable the legacy REST API".

[STEP 2: SET PERMALINKS]
Go to: Settings > Permalinks.
Action: Select "Post name" and save.

[STEP 3: GENERATE API KEYS]
Go to: WooCommerce > Settings > Advanced > REST API.
Action: Create a Key with "Read/Write" permissions.
Result: Copy Consumer Key/Secret into 'frontend/.env'.

[STEP 4: LOCALHOST PERMISSION FIX]
If you are working on Localhost, WooCommerce might block your 
React app. Follow these steps:
1. Open WPCode (Code Snippets).
2. Add a new PHP Snippet with the following code:

   add_filter( 'woocommerce_rest_check_permissions', function( $p, $c, $o, $pt ) {
       return true; 
   }, 99, 4 );

3. Activate the snippet.
NOTE: DEACTIVATE THIS SNIPPET BEFORE MOVING TO PRODUCTION/LIVE.
